## 简介
一个简单的Pytorch编写的集成学习文本情感分类项目  
使用 [sgns.weibo.bigram-char](https://github.com/Embedding/Chinese-Word-Vectors "@Chinese Word Vectors ") 预训练词向量
## 代码包含
1.加权投票集成  
2.基于Attention的集成  
3.基于Self-Attention的集成
